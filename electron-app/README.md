# OPAP Lottery Electron application

To start use `node` version `22.14.0` with `npm`: `10.9.2`.

## Quick Start Guide

```bash
# Clone the repository
git clone https://gitlab.dc.opap/deveng/janus/ssbt-kino.git

# Scripts for start the development server

# In a terminal tab, navigate to vue-app directory
cd vue-app
npm i
npm start

# In a new terminal tab, navigate to electron-app directory
cd electron-app
npm i
npm start
```

For more details see this wiki page: [OPAP Lottery Electron application](https://gitlab.dc.opap/deveng/janus/ssbt-kino/-/wikis/home/opap-lottery-electron-app)
